# Bob Coughlen Website Mockup

Static HTML/CSS website ready for Netlify.

## Deploy on Netlify

1. Upload this folder or the ZIP file to Netlify.
2. Netlify will serve `index.html` automatically.
3. The contact form uses Netlify Forms with `data-netlify="true"`.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `assets/` actual provided campaign images
