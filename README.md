# Bob Coughlen Campaign Website

Netlify-ready static website.

## How to upload to Netlify

1. Unzip this folder.
2. Replace the SVG placeholder images inside `/assets` with Bob's real photos if desired.
3. Drag the entire folder into Netlify Drop, or connect it to GitHub.
4. No build command is needed.
5. Publish directory should be the root folder.

## Edit these first

- `index.html` — content, events, phone/email, social links, donation link
- `styles.css` — colors, spacing, fonts
- `/assets` — replace placeholder artwork/photos

## Netlify form

The contact form already includes `data-netlify="true"`. After deployment, Netlify should detect it automatically.
