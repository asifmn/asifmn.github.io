# Creative Handyman Website

Ready-to-upload static website for Netlify.

## Deploy
1. Unzip the folder.
2. Drag the entire `creative-handyman-website` folder into Netlify Drop, or connect it to GitHub.
3. After Netlify assigns a URL, replace this placeholder everywhere:
   `https://creative-handyman-sartell.netlify.app`
   with the final domain.
4. Resubmit `sitemap.xml` in Google Search Console.
5. Connect the final website URL to the Google Business Profile.
